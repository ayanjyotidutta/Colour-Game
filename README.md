# OpticAI 👁️
**Precision Color Perception & Visual Memory Challenge**

OpticAI (subtly reading as 'Optical') is a high-fidelity web application designed to benchmark human color perception and short-term visual memory. Developed as a bridge between brand strategy and technical execution, it challenges users to match complex HSL spectrums under high-velocity constraints.

### 🎯 The Product Vision
As a Brand Manager, I wanted to explore the intersection of **Computer Vision** and **Human Perception**. OpticAI gamifies the "Munsell Color System" principles to test how accurately we retain brand visual equity (color) under pressure.

### 🛠️ Key Features
* **High-Precision Scoring:** A custom-engineered algorithm using a 1.8x cubic decay for perceptual accuracy.
* **The "Pressure Cooker" UX:** 5-second millisecond-precision memory phase followed by a 30-second recreation window.
* **Adaptive Retention Logic:** Dynamic "Try Beating X" goals based on performance to drive user replayability.
* **Industrial Aesthetic:** A minimalist, "Old Money" dark-mode UI with Glassmorphism and heartbeat-sync animations.

### 💻 Tech Stack
* **Framework:** React 18 (Functional Components & Hooks)
* **Styling:** Tailwind CSS (Utility-first JIT)
* **Logic:** Custom HSL-to-RGB conversion & Euclidean distance-based scoring.
* **Deployment:** Vercel / GitHub Actions.
